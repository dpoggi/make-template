# make-template

Project template for GNU make.

## `main.c`

Implements `pg_escape`, a tool that uses libpq to escape the string given as
its first argument.

## Getting Started

On macOS, do this first:

```sh
brew install makedepend pkg-config
```

Then:

```sh
make help
```
